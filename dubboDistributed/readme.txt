this is a distrubuted web system writed by rpc service framework dubbo.
the dubbo is run in spring boot technology.
credit-api is the interface of credit, credit-service is the implement class,and it is a provider.
inventory-api is the interface of inventory, inventory-service is the implement class,and it is a provider.
wms-api is the interface of watehouse management system, wms-service is the
implement class, and it is a provider.
order-service is a consumer.
